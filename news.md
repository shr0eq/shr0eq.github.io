  <h3>Created a sequential pulse switching measurement program using Python</h3>
  <div class="figure-row">
    <figure><img src="/assets/images/news/2025-08-10-a.png" alt="Pulse switching illustration" /></figure>
    <figure><img src="/assets/images/news/2025-08-10-b.png" alt="Pulse switching program" /></figure>
  </div>
  <p>I developed a sequential magnetization switching program using the Keithley 6221 and 2182A. To verify pulse-induced deterministic switching, applying sequential pulse currents with alternating polarity and measuring the anomalous Hall resistance is an effective approach. With this program, I can control the amplitude, width, and interval of the pulses. Additionally, I can configure the sequence by specifying the positive or negative polarity and preview the pulse patterns before starting a measurement.</p>
</div>

<div class="news-item" id="n-2025-05-08">
  <span class="news-date">2025-05-08</span>
  <h3>Created a new I–V curve measurement program using Python</h3>
  <div class="figure-row">
    <figure><img src="/assets/images/news/2025-05-08-a.jpg" alt="I-V curve program display" /></figure>
    <figure><img src="/assets/images/news/2025-05-08-b.png" alt="I-V curve program screenshot" /></figure>
  </div>
  <p>I created a new I–V curve measurement program using Python. Although it could have been easily written with LabVIEW, the purpose of this project was to completely write a measurement program using Python and AI, as LabVIEW, being a GUI language, is not easy to utilize with AI. The program's function is not to measure DC current I–V curves, but to apply desired AC currents and collect the corresponding 1st &amp; 2nd harmonic signals. I wrote it to perform forward &amp; backward sweeps within the set current range and to allow setting the desired frequency. AI isn't yet able to write a perfect program all at once, but I think there's significance in it helping to establish the framework for parts that were quite cumbersome and inconvenient. After modifying a few minor aspects, I was able to obtain a program that works very successfully.</p>
</div>
